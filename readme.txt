Make beautiful looking ONTRAPORT smartforms with a simple shortcode!

== Description ==

This plugin makes it easy for everyone to make their ONTRAPORT Smartforms looking professional and clean.

This works for both Office Autopilot 2.4 and Ontraport 3.0

== Installation ==

1. Upload `theme-my-ontraport-smartform` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
